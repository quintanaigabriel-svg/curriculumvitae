TP1 - Estructura HTML5

1 Este trabajo consiste en el desarrollo de un Curriculum Vitae utilizando HTML5 y CSS. Se aplicarán etiquetas semánticas para estructurar la página y estilos para su diseño visual. Se utilizó Flexbox para organizar los elementos en la página, logrando una mejor distribución del contenido.

2 El proyecto cuenta con:

Un archivo HTML principal (index.html)
Una página adicional de experiencia y estudios.
Una hoja de estilos externa (estilos.css)
Una carpeta "img" con las imágenes utilizadas -Una carpeta "fonts" con las tipografías utilizadas
3 Editor de código: Visual Studio Code
